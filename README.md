# Express Full-Stack App

## How to use

1. Install dependences
```bash
npm install
# or
npm i
```

2. Run the application

```bash
# Normal mode
npm run start

# Watch mode
npm run start:watch
```

## Run the tests
The tests are not configured, but if you configure them, you can run them with:

```bash
# Normal mode
npm run test

# Watch mode
npm run test:watch
```
